## ðŸ” INTERPRETASI SETIAP METRIC

### ACCURACY (AKURASI)
**Apa itu?** 
Persentase prediksi yang benar dari total prediksi

**Rumus:**
```
Accuracy = (Benar Positif + Benar Negatif) / Total Prediksi
```

**Contoh Kopi Nako (SVM):**
```
97.74% accuracy berarti:
- Test set: 354 reviews
- Prediksi benar: 346 reviews
- Prediksi salah: 8 reviews
```

**Kapan penting?** 
- Ketika semua jenis error sama pentingnya
- Overall model performance

---

### PRECISION (PRESISI)
**Apa itu?**
Dari semua review yang diprediksi POSITIF, berapa persen yang benar-benar positif?

**Rumus:**
```
Precision = Benar Positif / (Benar Positif + Salah Positif)
```

**Contoh Kopi Nako (SVM):**
```
97.84% precision berarti:
- Ketika model predict "POSITIF", 97.84% memang positif
- Hanya 0.16% diprediksi positif padahal negatif
```

**Kapan penting?**
- Ketika cost dari False Positive tinggi
- Contoh: Email spam detection (jangan sampai email penting masuk spam)

---

### RECALL (SENSITIVITY/TINGKAT PENEMUAN)
**Apa itu?**
Dari semua review yang benar-benar POSITIF, berapa persen yang terdeteksi model?

**Rumus:**
```
Recall = Benar Positif / (Benar Positif + Salah Negatif)
```

**Contoh Kopi Nako (SVM):**
```
97.74% recall berarti:
- Model berhasil mendeteksi 97.74% dari semua positive reviews
- Hanya miss 2.26% positive reviews
```

**Kapan penting?**
- Ketika cost dari False Negative tinggi
- Contoh: Disease detection (jangan miss patient dengan disease)

---

### F1-SCORE (HARMONIC MEAN)
**Apa itu?**
Balanced score antara Precision dan Recall

**Rumus:**
```
F1-Score = 2 Ã— (Precision Ã— Recall) / (Precision + Recall)
```

**Contoh Kopi Nako (SVM):**
```
97.74% F1-Score berarti:
- Perfect balance antara precision (97.84%) dan recall (97.74%)
- Model sangat baik di kedua aspek
```

**Kapan penting?**
- **PALING PENTING** untuk imbalanced datasets
- Overall model evaluation
- Ketika precision dan recall sama pentingnya

**Mengapa F1-Score terpenting untuk sentiment analysis?**
- Kami ingin catch semua negative reviews (high recall)
- Tapi juga harus akurat (high precision)
- F1-Score measure keduanya dalam satu angka
